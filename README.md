# MindLog — Inteligência em Bem-Estar Emocional

O **MindLog** é uma plataforma SaaS premium dedicada à saúde mental e ao autoconhecimento, unindo tecnologia de IA empática com uma interface acolhedora e moderna.

## ✨ Funcionalidades

- 🧠 **IA Terapêutica**: Conversas em tempo real com apoio emocional contínuo.
- 📓 **Diário Inteligente**: Espaço seguro para registros diários com análise de sentimentos.
- 📊 **Insights Profundos**: Gráficos e tendências baseadas no seu humor e relatos.
- 🧘 **Exercícios Guiados**: Práticas de respiração e meditação personalizadas.
- 👥 **Comunidade Segura**: Feed social moderado para compartilhamento de experiências.
- 🛡️ **Painel Admin**: Gestão completa de usuários e métricas do sistema.

## 🚀 Tecnologias

- **Next.js 14** (App Router)
- **TypeScript**
- **Vanilla CSS** (Custom Design System)
- **Lucide React** (Ícones)
- **Framer Motion** (Animações)

## 🛠️ Como Executar

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
